OOR API Documentation

Welcome to the Reseller Integration Guide. Use this API to authenticate Smart TVs directly from your own website.

💡 Fair Quota System: Your quota points are only deducted on a successful login. If a user enters a wrong code or an account is dead, you do not lose any points!
⚠️ Security Notice: Never expose your API key in frontend JavaScript or HTML. You must make API requests from your own backend server (Node.js, PHP, Python, etc.) to keep your key hidden.

Authentication Endpoint

POST /api/tv-login

Required Headers

HeaderRequiredDescription
X-API-KeyYESYour secret Reseller API Key.
OriginYESThe domain authorized for your key (e.g., https://your-website.com). Automatically sent by browsers, but must be set manually if calling from a backend proxy.

Form Data Parameters

Data must be sent as multipart/form-data or application/x-www-form-urlencoded.

ParameterRequiredDescription
tv_codeYESThe 8-digit TV code from the screen.
planNOFilter by plan (e.g., "Premium", "Standard").
qualityNOFilter by quality (e.g., "UHD", "HD").
languageNOFilter by language (e.g., "English", "Spanish").

Integration Examples

PHP (Easiest Backend Method) 
<?php
// Save this as tv-login.php on your server
$api_key = "oor_YOUR_API_KEY";
$api_url = "https://YOUR_API_DOMAIN/api/tv-login";
$your_domain = "https://your-website.com"; // Must match your authorized domain

$postData = [
    'tv_code'  => $_POST['tv_code'] ?? '',
    'plan'     => $_POST['plan'] ?? '',       // Optional
    'quality'  => $_POST['quality'] ?? '',    // Optional
    'language' => $_POST['language'] ?? ''    // Optional
];

$ch = curl_init($api_url);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_POSTFIELDS, http_build_query($postData));
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    "X-API-Key: " . $api_key,
    "Origin: " . $your_domain
]);

$response = curl_exec($ch);
curl_close($ch);

header('Content-Type: application/json');
echo $response;
?>
Node.js / Express (Using Axios) 
const axios = require('axios');
const FormData = require('form-data');

app.post('/my-tv-login', async (req, res) => {
    const form = new FormData();
    form.append('tv_code', req.body.tv_code);
    if (req.body.plan) form.append('plan', req.body.plan);

    try {
        const response = await axios.post('https://YOUR_API_DOMAIN/api/tv-login', form, {
            headers: {
                ...form.getHeaders(),
                'X-API-Key': 'oor_YOUR_API_KEY',
                'Origin': 'https://your-website.com'
            }
        });
        res.json(response.data);
    } catch (error) {
        res.status(error.response?.status || 500).json(error.response?.data);
    }
});
Python (Requests) 
import requests

url = "https://YOUR_API_DOMAIN/api/tv-login"
payload = {
    "tv_code": "12345678",
    "plan": "Premium",
    "quality": "UHD"
}
headers = {
    "X-API-Key": "oor_YOUR_API_KEY",
    "Origin": "https://your-website.com"
}

response = requests.post(url, data=payload, headers=headers)
print(response.json())
