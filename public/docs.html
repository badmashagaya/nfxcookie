OOR API Documentation

Welcome to the Reseller Integration Guide. Use this documentation to seamlessly connect your store or web application to the OOR dynamic authentication engine.

💡 Fair Quota System: Your quota limit is strictly protected. Points are only deducted on a fully successful login. If a user enters an incorrect TV code, or if an account fails to link, you are not charged any usage!
⚠️ Security Architecture & Key Protection

Never expose your X-API-Key in frontend HTML, Vanilla JavaScript, React, or mobile applications. If placed on the client-side, anyone can steal your key and drain your quota.

Correct Flow:
1. Your user submits the TV code on your website's frontend.
2. Your frontend sends the code to your own backend server (Node.js, PHP, Python).
3. Your backend server securely attaches your API key and forwards the request to the OOR API.
4. Your backend returns the result back to the user.

Authentication Endpoint

This endpoint processes the 8-digit TV code and attempts to authenticate the user using the available database pool based on your applied filters.

POST https://your-api-domain.com/api/tv-login

Required Headers

HeaderRequiredDescription
X-API-KeyYESYour secret Reseller API Key provided by the admin.

Form Data Parameters

Payload data must be sent exactly as multipart/form-data or application/x-www-form-urlencoded.

ParameterRequiredDescription
tv_codeYESThe 8-digit TV rendezvous code displayed on the user's screen.
planNOFilter the account pool by plan (e.g., "Premium", "Standard").
qualityNOFilter the account pool by quality (e.g., "UHD", "HD").
languageNOFilter the account pool by display language (e.g., "English").

API Responses & Error Codes

The API returns standard HTTP status codes to indicate the success or failure of the request.

HTTP StatusMeaningDetails
200 OKProcessedThe request was processed. Check the success boolean in the JSON body to see if the TV code was accepted or rejected.
401 UnauthorizedAuth ErrorMissing or Invalid X-API-Key header.
403 ForbiddenDeactivatedYour API Key has been suspended or deactivated by the admin.

Integration Examples

Below are clear backend examples of how to safely construct the HTTP request to the OOR API.

Node.js / Express (Native Fetch) 
// 1. Setup your secure backend route
app.post('/api/authenticate-tv', async (req, res) => {
    
    // 2. Prepare the Form Data payload
    const formData = new URLSearchParams();
    formData.append('tv_code', req.body.tv_code);
    if (req.body.plan) formData.append('plan', req.body.plan);

    try {
        // 3. Securely forward to OOR
        const response = await fetch('https://YOUR_API_DOMAIN/api/tv-login', {
            method: 'POST',
            headers: {
                'X-API-Key': 'oor_YOUR_HIDDEN_API_KEY',
                'Content-Type': 'application/x-www-form-urlencoded'
            },
            body: formData.toString()
        });

        const data = await response.json();
        res.status(response.status).json(data);

    } catch (error) {
        res.status(500).json({ success: false, message: "Internal Server Error" });
    }
});
PHP (cURL) 
<?php
$api_key = "oor_YOUR_HIDDEN_API_KEY";
$api_url = "https://YOUR_API_DOMAIN/api/tv-login";

// Safely map user input to POST payload
$postData = [
    'tv_code'  => $_POST['tv_code'] ?? '',
    'plan'     => $_POST['plan'] ?? ''
];

$ch = curl_init($api_url);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_POST, true);
curl_setopt($ch, CURLOPT_POSTFIELDS, http_build_query($postData));
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    "X-API-Key: " . $api_key,
    "Content-Type: application/x-www-form-urlencoded"
]);

$response = curl_exec($ch);
$httpcode = curl_getinfo($ch, CURLINFO_HTTP_CODE);
curl_close($ch);

header('Content-Type: application/json');
http_response_code($httpcode);
echo $response;
?>
Python (Requests) 
import requests

api_key = "oor_YOUR_HIDDEN_API_KEY"
api_url = "https://YOUR_API_DOMAIN/api/tv-login"

headers = {
    "X-API-Key": api_key
}

payload = {
    "tv_code": "12345678",
    "plan": "Premium"
}

response = requests.post(api_url, data=payload, headers=headers)

print(f"Status Code: {response.status_code}")
print(response.json())
