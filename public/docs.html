OOR API Documentation

Welcome to the Reseller Integration Guide. Use this documentation to securely connect your website to the OOR authentication engine.

💡 Fair Quota System: Your quota points are only deducted on a successful login. If a user enters a wrong code or an account fails, you do not lose any points!
⚠️ Security Notice: Never expose your API key in frontend HTML or JavaScript. You must make API requests from your own backend server to keep your key completely hidden.

Authentication Endpoint

Send your requests to this URL.

POST https://your-api-domain.com/api/tv-login

Required Headers

HeaderRequiredDescription
X-API-KeyYESYour secret Reseller API Key.
OriginYESThe domain authorized for your key (e.g., https://your-website.com). This proves the request is coming from your server.

Form Data Parameters

Data must be sent as multipart/form-data. Do not send raw JSON bodies.

ParameterRequiredDescription
tv_codeYESThe 8-digit TV code from the screen.
planNOFilter by plan (e.g., "Premium", "Standard"). Leave blank to ignore.
qualityNOFilter by quality (e.g., "UHD", "HD"). Leave blank to ignore.
languageNOFilter by language (e.g., "English"). Leave blank to ignore.

Integration Examples

Below are clear, copy-paste examples of how to build your backend endpoint. Your website's frontend will call your own backend, and your backend will securely forward the request to OOR.

Node.js (Using built-in Native Fetch) 
// 1. Your user submits the form on your website
app.post('/my-tv-login', async (req, res) => {
    
    // 2. Prepare the data to forward to OOR
    const formData = new FormData();
    formData.append('tv_code', req.body.tv_code);
    
    // Optional filters from your form
    if (req.body.plan) formData.append('plan', req.body.plan);
    if (req.body.quality) formData.append('quality', req.body.quality);

    try {
        // 3. Send secure request to OOR API
        const response = await fetch('https://YOUR_API_DOMAIN/api/tv-login', {
            method: 'POST',
            headers: {
                'X-API-Key': 'oor_YOUR_HIDDEN_API_KEY',
                'Origin': 'https://your-website.com' // Must match your allowed domain
            },
            body: formData
        });

        const data = await response.json();
        
        // 4. Return the OOR success/fail JSON to your frontend
        res.status(response.status).json(data);

    } catch (error) {
        res.status(500).json({ success: false, message: "Server error" });
    }
});
PHP (cURL) 
<?php
// Save this as tv-login.php on your server
$api_key = "oor_YOUR_HIDDEN_API_KEY";
$api_url = "https://YOUR_API_DOMAIN/api/tv-login";
$your_domain = "https://your-website.com";

$postData = [
    'tv_code'  => $_POST['tv_code'] ?? '',
    'plan'     => $_POST['plan'] ?? '',       
    'quality'  => $_POST['quality'] ?? '',    
    'language' => $_POST['language'] ?? ''    
];

$ch = curl_init($api_url);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_POSTFIELDS, $postData);
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    "X-API-Key: " . $api_key,
    "Origin: " . $your_domain
]);

$response = curl_exec($ch);
$httpcode = curl_getinfo($ch, CURLINFO_HTTP_CODE);
curl_close($ch);

header('Content-Type: application/json');
http_response_code($httpcode);
echo $response;
?>
